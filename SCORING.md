# 풀업 점수 가이드라인

## 목적

이 문서는 풀업 분석기의 `누적 Score` 기준을 정리한 문서다.

## 레벨 구간

경계 점수는 상위 레벨에 포함한다.


| 시작 점수   | 종료 점수   | 레벨  | 비고              |
| ------- | ------- | --- | --------------- |
| `0점`    | `999점`  | 비기너 | 기본 체력 형성 구간     |
| `1000점` | `1999점` | 중급자 | 안정적인 누적 수행 구간   |
| `2000점` | `2999점` | 고급자 | 높은 품질의 반복 누적 구간 |
| `3000점` | `4999점` | 마스터 | 상위권 수행 누적 구간    |
| `5000점` | 이상      | 신   | 최상위 수행 구간       |


## 1회 점수 구성


| 항목     | 기준            | 점수 반영      | 설명                       |                  |
| ------ | ------------- | ---------- | ------------------------ | ---------------- |
| 기본 점수  | rep 1회 인정     | `+90점`     | 1회 수행에 대한 기본 적립 점수       |                  |
| 중심 안정성 | 중심 trajectory | `-0 ~ 5점`  | 좌우 흔들림이 클수록 감점           |                  |
| 높이     | 바 대비 최고 도달 높이 | `-10 ~ +10점` | 기준 높이 미달 시 감점, 기준 이상이면 가점 |                  |
| 각도     | 팔 수축 품질       | `-10 ~ +10점` | 기준 각도 미달 시 감점, 깊게 수축하면 가점 |                  |
| 데드행    | rep 시작 자세     | `+10점`     | 완전 신전 시작 시 보너스           |                  |
| 탑홀딩    | 고점 근처 정지 시간   | `+0 ~ 50점` | 높이와 각도 기준을 동시에 만족한 상태에서 `0.1초당 5점`, 최대 `1초` |                  |
| 최종 점수  | 1회 최종 점수      | `0점 이상`    | 좋은 자세와 홀딩이면 `100점 초과 가능` |                  |


## 1회 점수 산식

```text
Rep Score
= max(
    0,
    90
    - center_penalty(0~5)
    + height_adjustment(-10~+10)
    + angle_adjustment(-10~+10)
    + deadhang_bonus(0 or 10)
    + top_hold_bonus(0~50)
  )
```

## 총점 반영 방식


| 항목        | 적용 방식                          |
| --------- | ------------------------------ |
| 총점        | 모든 rep 점수를 누적 합산               |
| HUD       | 현재 누적 `Score`와 `Level` 표시      |
| 점수 바      | 좌측 세로 막대가 누적 점수에 따라 아래에서 위로 상승 |
| rep 점수 알림 | rep 완료 시 막대 옆에 `+80` 형태로 표시    |
