# 풀업 점수 가이드라인

## 목적

이 문서는 풀업 분석기의 `누적 Score` 기준을 정리한 문서다.

## 레벨 구간

경계 점수는 상위 레벨에 포함한다.

| 시작 점수 | 종료 점수 | 레벨 | 비고 |
| --- | --- | --- | --- |
| `0점` | `799점` | 초급자 | 기본 체력 형성 구간 |
| `800점` | `1599점` | 중급자 | 안정적인 누적 수행 구간 |
| `1600점` | `2399점` | 고급자 | 높은 품질의 반복 누적 구간 |
| `2400점` | `3999점` | 마스터 | 상위권 수행 누적 구간 |
| `4000점` | 이상 | 신 | 최상위 수행 구간 |

## 1회 점수 구성

| 항목 | 기준 | 점수 반영 | 설명 |
| --- | --- | --- | --- |
| 기본 점수 | rep 1회 인정 | `+90점` | 1회 수행에 대한 기본 적립 점수 |
| 중심 안정성 | 중심 trajectory | `-0 ~ 5점` | 좌우 흔들림이 클수록 감점 |
| 템포 | 반복 리듬 일관성 | `-0 ~ 5점` | 리듬이 무너지면 감점 |
| 높이 | 바 대비 최고 도달 높이 | `-0 ~ 10점` | 기준 높이 미달 시 강한 감점 |
| 각도 | 팔 수축 품질 | `-0 ~ 10점` | 기준 각도 미달 시 강한 감점 |
| 데드행 | rep 시작 자세 | `+10점` | 완전 신전 시작 시 보너스 |
| 최종 점수 제한 | 1회 최종 점수 | `0 ~ 100점` | 계산 결과를 100점 기준으로 제한 |

## 1회 점수 산식

```text
Rep Score
= clamp(
    90
    - center_penalty(0~5)
    - tempo_penalty(0~5)
    - height_penalty(0~10)
    - angle_penalty(0~10)
    + deadhang_bonus(0 or 10),
    0,
    100
  )
```

## 총점 반영 방식

| 항목 | 적용 방식 |
| --- | --- |
| 총점 | 모든 rep 점수를 누적 합산 |
| HUD | 현재 누적 `Score`와 `Level` 표시 |
| 그래프 | `SCORE TREND`에 시간 대비 누적 점수 표시 |
| rep 점수 알림 | rep 완료 시 `+80` 형태로 화면에 표시 |
